Tag + Search

Organize & Find

tag

Tagged a1b2c3 with: genetics
search by tag
$ pubsave search -t genetics

2 papers found
search by keyword
$ pubsave search leprosy

1 paper found
search by author
$ pubsave search -a Zhang

2 papers found
